# I-TEXGEO Data Management Plan (DMP)

This repository contains the **Data Management Plan (DMP)** for the Horizon Europe project:

- **Project Acronym:** I-TEXGEO  
- **Action Number:** 101235387  
- **Title:** *IoT Supported Electronic Geotextiles for Sustainable and Smart Precision Agriculture*  
- **DMP Version:** 1.0  
- **Date:** February 6, 2026  

## Contents

- `dmp.pdf` — Official Data Management Plan document.

## Summary

The DMP describes the project’s approach to managing research data in compliance with **Horizon Europe Open Science requirements** and the **FAIR principles** (Findable, Accessible, Interoperable, Reusable). It outlines expected data types, storage formats, metadata practices, access conditions, long-term preservation, and data security measures.
